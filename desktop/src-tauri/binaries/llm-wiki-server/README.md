Place LLM Wiki sidecar binaries under platform directories:

- `darwin-arm64/llm-wiki-server`
- `darwin-amd64/llm-wiki-server`
- `linux-arm64/llm-wiki-server`
- `linux-amd64/llm-wiki-server`
- `windows-amd64/llm-wiki-server.exe`

Build them from `llm_wiki/src-tauri` with the `llm-wiki-server` binary target.
